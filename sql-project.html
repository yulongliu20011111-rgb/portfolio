Game Launch ROI Analysis: Seasun Games

Analyzing influencer marketing performance on Douyin (TikTok China) and Rednote (Xiaohongshu) to optimize user acquisition for Kingsoft Seasun Games.

1. The Business Challenge

During a major title update for Kingsoft Seasun Games, the marketing team executed a cross-platform influencer campaign. The objective was to identify which content creators delivered the most cost-effective Engagement Rate and Cost Per Engagement (CPE) to establish a baseline for future game publishing budgets.

Key Metrics Calculated

  • Weighted Engagement Score
  • Cost Per Engagement (CPE)
  • Cross-Platform Efficiency

Tech Stack

  • PostgreSQL
  • Window Functions (DENSE_RANK)
  • CTEs (Common Table Expressions)

2. Data Realities: Messy Data & Trade-offs

  • Handling Incomplete Data: Engagement logs from the Douyin API contained a 5% null rate due to server timeouts during peak launch hours. I addressed this by applying a rolling 7-day average imputation for missing hourly data to maintain overall aggregate accuracy.
  • Metric Trade-offs: I assumed a uniform 1:1:1 weighting for likes, comments, and shares across platforms. While a Rednote "save" (often used for game guides/builds) typically indicates higher long-term player retention intent than a Douyin "like", standardizing the weights allowed for a rapid, baseline cross-platform comparison in this sprint.
  • Attribution Limitations: The analysis relies on a strict 14-day post-click attribution window. This limitation means the long-tail organic community growth generated by high-quality fan-art and lore explanations on Rednote may be slightly undercounted.

3. The SQL Execution

PostgreSQL 
-- Step 1: Aggregate engagement metrics per influencer & platform
WITH Engagement_Summary AS (
    SELECT 
        i.influencer_name,
        i.content_category,
        i.platform,
        c.budget_allocation,
        SUM(p.views) AS total_views,
        SUM(p.likes + p.comments + p.shares) AS total_engagements
    FROM influencers i
    JOIN posts p ON i.influencer_id = p.influencer_id
    JOIN campaigns c ON p.campaign_id = c.campaign_id
    WHERE c.campaign_name = 'Seasun_Q3_Expansion'
    GROUP BY 1, 2, 3, 4
),

-- Step 2: Calculate KPI ratios (Engagement Rate & CPE)
Performance_Calc AS (
    SELECT
        *,
        ROUND((total_engagements::numeric / NULLIF(total_views,0)) * 100, 2) AS engagement_rate,
        ROUND(budget_allocation / NULLIF(total_engagements,0), 2) AS cost_per_engagement
    FROM Engagement_Summary
)

-- Step 3: Rank top performers by Platform efficiency
SELECT 
    influencer_name,
    content_category
