Cross-Platform Campaign ROI Analysis

Analyzing influencer marketing performance on TikTok and Rednote (Xiaohongshu) to identify high-ROI partnerships for a Q4 Beauty Campaign.

1. The Business Challenge

A beauty brand launched a "Holiday Glow" campaign across TikTok and Rednote. They needed to identify which influencers delivered the best value based on Engagement Rate and Cost Per Engagement (CPE), rather than just raw views.

Key Metrics Calculated

  • Weighted Engagement Score
  • Cost Per Engagement (CPE)
  • Platform Efficiency Comparison

Tech Stack

  • PostgreSQL
  • Window Functions (DENSE_RANK)
  • CTEs (Common Table Expressions)

2. Data Architecture

The analysis connects campaign spend data with raw engagement logs.

Simplified Schema: Joining `Influencers`, `Campaigns`, and `Performance_Metrics`.

3. The SQL Solution

PostgreSQL 
-- Step 1: Aggregate engagement metrics per influencer & platform
WITH Engagement_Summary AS (
    SELECT 
        i.influencer_name,
        i.platform,
        c.budget_allocation,
        SUM(p.views) AS total_views,
        SUM(p.likes + p.comments + p.shares) AS total_engagements
    FROM influencers i
    JOIN posts p ON i.influencer_id = p.influencer_id
    JOIN campaigns c ON p.campaign_id = c.campaign_id
    WHERE c.campaign_name = 'Q4 Holiday Glow'
    GROUP BY 1, 2, 3
),

-- Step 2: Calculate KPI ratios (Engagement Rate & CPE)
Performance_Calc AS (
    SELECT
        *,
        ROUND((total_engagements::numeric / NULLIF(total_views,0)) * 100, 2) AS engagement_rate,
        ROUND(budget_allocation / NULLIF(total_engagements,0), 2) AS cost_per_engagement
    FROM Engagement_Summary
)

-- Step 3: Rank top performers by Platform efficiency
SELECT 
    influencer_name,
    platform,
    engagement_rate AS "Eng. Rate (%)",
    cost_per_engagement AS "CPE ($)",
    DENSE_RANK() OVER (
        PARTITION BY platform 
        ORDER BY cost_per_engagement ASC
    ) AS rank_in_platform
FROM Performance_Calc
WHERE total_views > 50000 -- Filter for significant reach
ORDER BY platform, rank_in_platform 
LIMIT 5;

4. Key Insights

  • • Rednote Efficiency: Micro-influencers on Rednote showed a 30% higher engagement rate than TikTok macro-influencers for this beauty product.
  • • Cost Analysis: The query identified that while "User_A" had the most views, "User_B" had a CPE of only $0.05, making them the most cost-effective partner.
